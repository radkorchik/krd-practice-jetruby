## Отчеты по практике (26.03 - 18.04)

### 26.03
Анализ требований курсового проекта. Проектирование структуры сущностей: пользователи, посты, подписки, комментарии, лайки. Подготовка плана разработки по этапам.

### 27.03
Создание нового Rails-приложения `instagram-clone`. Проверка базовой структуры проекта, конфигурации окружений и маршрутизации.

### 28.03
Подключение Devise для регистрации и аутентификации. Генерация модели пользователя и базовой конфигурации входа/выхода.

### 29.03
Настройка RSpec, Shoulda Matchers, Capybara и DatabaseCleaner. Подготовка тестовой инфраструктуры проекта.

### 30.03
Проектирование и создание миграций для Post, Comment, Follow, Like. Добавление индексов и ограничений целостности в базе данных.

### 31.03
Настройка Active Storage для загрузки изображений. Проверка сохранения вложений и связи изображений с постами.

### 01.04
Реализация моделей и ассоциаций между сущностями. Добавление валидаций бизнес-правил для подписок, лайков и комментариев.

### 02.04
Реализация контроллера постов: создание, просмотр, редактирование, удаление. Настройка ограничений доступа на редактирование только владельцем.

### 03.04
Реализация страниц отображения постов и формы публикации. Добавление вывода изображений и счетчика лайков.

### 04.04
Реализация подписок и отписок между пользователями. Добавление защиты от подписки на самого себя.

### 05.04
Реализация страницы списка пользователей и профиля. Добавление кнопок управления подпиской из интерфейса.

### 06.04
Реализация пользовательской ленты на основе подписок. Подключение сортировки по дате публикации.

### 07.04
Реализация добавления комментариев к постам. Настройка удаления только собственных комментариев.

### 08.04
Реализация механики like/dislike через toggle. Добавление отображения количества лайков на странице поста.

### 09.04
Доработка layout и навигации приложения. Добавление flash-сообщений для обратной связи по действиям пользователя.

### 10.04
Кастомизация страниц регистрации и входа Devise. Добавление поля username и проверка корректной передачи параметров.

### 11.04
Написание модельных тестов для User, Post, Comment, Follow, Like. Проверка валидаций, ассоциаций и ограничений уникальности.

### 12.04
Написание request-спеков для постов и прав доступа. Проверка авторизации и сценариев владелец/не владелец.

### 13.04
Написание request-спеков для подписок, комментариев и лайков. Проверка ограничений доступа для неавторизованных пользователей.

### 14.04
Написание request-спеков для ленты и профилей пользователей. Проверка выборки постов в ленту по подпискам.

### 15.04
Написание системных тестов Capybara для сценариев регистрации, входа и публикации поста. Проверка основных пользовательских потоков.

### 16.04
Расширение системных тестов для подписок, комментариев и лайков. Проверка поведения интерфейса после пользовательских действий.

### 17.04
Отладка тестового набора и исправление ошибок сценариев. Повторный прогон полного тестового набора до зеленого состояния.

### 18.04
Финальная проверка функционала курсового проекта. Подготовка итоговой структуры проекта и фиксация прогресса по этапным коммитам.
