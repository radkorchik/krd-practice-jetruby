# Reports App

Веб-приложение на **Ruby on Rails** для учёта **лабораторных отчётов** студентов и выставления **оценок по шкале ECTS** (A, B, C, D, E, FX, F).

## Возможности

- Просмотр списка отчётов с указанием студента, заголовка и оценки.
- Создание, редактирование и удаление отчётов.
- Привязка каждого отчёта к пользователю (студенту).
- Валидации полей согласно требованиям задания (длины строк, формат email, допустимые оценки).

## Технологии

- Ruby on Rails 8.x
- SQLite (разработка)
- ERB-шаблоны, Propshaft

## Запуск

```bash
cd lesson-7/reports-app
bundle install
bin/rails db:migrate db:seed
bin/rails server
```

Открой в браузере: <http://localhost:3000>.

## Пользователи

Отдельный UI для регистрации студентов не обязателен по заданию. Пользователей можно создать через сиды (`db/seeds.rb`) или в консоли:

```bash
bin/rails console
```

```ruby
User.create!(first_name: "Имя", last_name: "Фамилия", email: "student@example.com")
```

## Структура данных

- **users** — имя, фамилия, email.
- **lab_reports** — заголовок, описание, оценка ECTS, связь с пользователем (`user_id`).
